# 起步

## 环境所需工具检查

- 检查用户是否安装了 node.js，没有的话需要先安装 node.js（至少 18.20 版本以上）。
- 检查用户是否安装了 git 工具，没有的话需要安装
- 查看用户本地是否安装了 kingdee-cosmic-cli
- 检查用户是否安装了 pnpm，没有的话需要安装

kingdee-cosmic-cli 的检查可以查看版本：

```bash
cosmic --version
```

没有，可以安装：

```bash
npm install -g kingdee-cosmic-cli
```

如果已经存在，**一定要先更新到最新版本**：

```bash
npm update -g kingdee-cosmic-cli
```

## 新建项目

### 用户提问场景预测 1

如果用户只说要创建一个自定义控件项目，你需要先问他几个问题：

1. 自定义控件的名称是什么？也就是要注册在设计器中的控件名称。名称必须是英文，并且推荐 `xxx-xxx` 命名方式。
2. 想用什么前端 UI 框架进行开发？React 还是 Vue？
3. 如果用户选择了用 React，继续追问是否集成 Antd UI 组件库。

执行脚本创建项目：

┌───────────────────────────────────┬──────────────────────────────────────┐
│ 命令 │ 说明 │
├───────────────────────────────────┼──────────────────────────────────────┤
│ cosmic create my-app --react │ 使用 React 模板（不集成 Ant Design） │
├───────────────────────────────────┼──────────────────────────────────────┤
│ cosmic create my-app --react-antd │ 使用 React + Ant Design 模板 │
├───────────────────────────────────┼──────────────────────────────────────┤
│ cosmic create my-app --vue │ 使用 Vue3 模板 │
├───────────────────────────────────┼──────────────────────────────────────┤
│ cosmic create my-app │ 不带选项时仍进入交互式问答模式 │
└───────────────────────────────────┴──────────────────────────────────────┘

### 用户提问场景预测 2

如果用户说要创建一个自定义控件项目，并且提供了需要的前端 UI 框架，初级需求内容等，你需要先帮忙执行脚本创建项目。

至于用户说的需求，可以等后面的[项目创建成功]完成后，有了 CLAUDE.md 文件再执行。

## 项目创建成功

### 依赖安装

项目创建完成后，帮用户使用 pnpm 安装依赖。如果用户没有安装 pnpm，提醒下用户，引导安装 pnpm。

### agent memory 文件拷贝

- 如果用户选用了 React，从 `/assets/react/` 里拷贝 CLAUDE.md 文件到项目根目录中。
- 如果用户选用了 Vue，从 `/assets/vue/` 里拷贝 CLAUDE.md 文件到项目根目录中。
